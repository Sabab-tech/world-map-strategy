# World Map Strategy Project

একটি নিখুঁত এবং ইন্টারেক্টিভ বিশ্ব মানচিত্র (Interactive World Map) প্রজেক্ট। এটি হাত দিয়ে না এঁকে সম্পূর্ণ কোডিং এবং রিয়েল-টাইম জিওগ্রাফিক্যাল ডেটা (GeoJSON) ব্যবহার করে তৈরি করা হচ্ছে।

## বৈশিষ্ট্যসমূহ (Features)
- **নিখুঁত সীমানা:** পৃথিবীর সব দেশের আসল সীমানা এবং বর্ডার স্বয়ংক্রিয়ভাবে লোড হবে।
- **জুম ও ড্র্যাগ:** মোবাইলের স্ক্রিনে চিমটি কেটে (Pinch) জুম করা এবং টেনে টেনে (Drag) পুরো পৃথিবী দেখা যাবে।
- **স্মার্ট রেসপন্সিভ:** যেকোনো মোবাইল বা কম্পিউটারের স্ক্রিনে ম্যাপটি সুন্দরভাবে ফিট হবে।
- **কোনো অ্যান্ড্রয়েড লক নেই:** সম্পূর্ণ ক্লাউড ভিত্তিক হওয়ায় মোবাইলের ইন্টারনাল স্টোরেজ পারমিশনের কোনো ঝামেলা নেই।

## প্রযুক্তি (Technologies Used)
- HTML5 & CSS3
- JavaScript
- Leaflet.js / D3.js (Geographical Mapping Libraries)
